# Excel-to-Dashboard React Assignment

## Project Overview
This project reads an Excel file and renders:
- Tabular data
- A chart visualization
- Search, sorting, and pagination features

## Tech Stack
- React (Vite)
- xlsx (SheetJS)
- Recharts

## Setup Instructions
1. npm install
2. npm run dev

## Excel File Structure
File: sample-data.xlsx
Sheet Name: Data
Columns:
- Month
- Region
- Sales
- Profit

## Chart Mapping
Bar chart displaying:
Sales by Month

## Features Implemented
- Auto-load Excel file
- Excel parsing
- Table rendering
- Search
- Sorting
- Pagination
- Error handling
- Responsive layout

## Deployment
Deployed on Vercel.
